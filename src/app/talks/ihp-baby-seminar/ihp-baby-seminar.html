Symplectic Billiard

Fix a convex body {{ "$K\\subset\\mathbb{R}^2$" }} The symplectic billiard maps the directed chord $xy$ to $yz$.

Symplectic Billiard Table Map

Question: given a convex $n$-gon $P$, can we construct $Q$ circumscribed s.t. $P$ is a closed symplectic billiard trajectory in $Q$?
Answer: yes!

Symplectic Billiard Table Map

$T(P)=Q$ $T([P])=[Q]$

Prior Art

  1. Pentagram map
    • Projectively invariant
    • Identity for $n=5$ and $n=6$
    • Integrable for all $n$ (Schwartz et al.)
  2. Usual billiard table map
    • Only commutes with rigid transformations
    • Converges to regular for every $n$ (Sharipova)

Experimental Results

Under the symplectic billiard table map, an $n$-gon tends towards...
  • $n=5$: regularity ($n=3,4$ trivial)
  • $n=6$: 3-fold rotational symmetry
  • $n\geq 7$: degeneracy ($k$-gon with {{ '$k\\leq\\left\\lfloor\\frac{n-1}{2}\\right\\rfloor$' }})
Hard to prove: I don't know of natural coordinates for {{ '$\\{n$-gons$\\}/\\mathrm{Aff}$' }}.

Twisted Polygons

Definition:

An (affine) twisted $n$-gon is a sequence of vertices

{{ '$(\\{\\dots,v_{-1},v_0,v_1\\dots\\},M)$' }}

where {{ '$M\\in\\mathrm{Aff}$' }} and {{ '$v_{n+i}=Mv_i$.' }}

Twisted Polygons

Remark:
  • {{ '$\\dim(\\{n$-gons$\\}/\\mathrm{Aff}) = 2n-6$.' }}
  • {{ '$\\dim(\\{$twisted $n$-gons$\\}) = 2n + 6$.' }}
  • {{ '$\\dim(\\{$twisted $n$-gons$\\}/\\mathrm{Aff}) = 2n + 6 - 6$.' }}
  • Larger space but no pesky relations!

Result for 2-gons

Theorem: When $n=2$, the map is super-integrable: there is an invariant Poisson bracket with three Poisson-commuting integrals:
  1. det($M$)
  2. tr($M$)
  3. a mysterious third one
