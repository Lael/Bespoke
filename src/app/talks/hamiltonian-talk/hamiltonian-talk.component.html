Behavior “at Infinity”

Theorem 1 (a)

Let {{ "$K\\in\\mathbb{R}^2$" }} be convex and let be $x$ be a non-singular point of the outerArea length billiard around $K$. If {{ "$\\mathcal{O}$" }} is the “once around” orbit of $x$, then {{ "$\\mathcal{O}$" }} is contained in an annulus of thickness $C_1$ depending only on $K$.

Simply put, distant orbits are approximately circular over bounded time intervals.

Proof Sketch

1. Replace table with line segment. 1. Replace table with line segment. 2. Lemma 1: $x$ and $T(x)$ lie on $E_1$. 3. Lemma 1: $T(x)$ and $T^2(x)$ lie on $E_2$. 4. Hausdorff distance from $E_1$ to $E_2$ ~ $|p_1r_2|+|p_2r_1|$. 5. Sum of radial displacements ~ perimeter of $K$. $\square$

Behavior “at Infinity”

Theorem 1 (b)

Suppose $K$ is smooth with positive curvature. Let {{ "$w(\\theta)$" }} be the width of $K$ as seen from angle {{ "$\\theta$" }} and let {{ "$ \\Omega = \\frac{r dr \\wedge d\\theta }{w(\\theta)} $" }}. If {{ "$H(r,\\theta)= 2r$" }} and {{ "$X(r,\\theta)= -\\frac{2w(\\theta)}{r} \\partial_\\theta$" }}, then {{ "$$\\big|T^2(x) - \\Phi(x) \\big| \\leq C_2/|x|, $$" }} where {{ "$\\Phi$" }} is the time-1 flow of $X$ and $C_2$ depends only on $K.$

Rate of Escape

Corollary 1

For any $K$ and non-singular $x$, {{ "$|T^n(x)| = O(\\sqrt{n}).$" }}

Idea: $k$-th trip around the origin increases radius by $O(1)$, but takes $O(k)$ steps, so it takes $n=O(k^2)$ steps to reach distance {{ "$k = O(\\sqrt{n})$" }}.

KAM-type Result

Corollary 2

For smooth enough $K$, there are distant invariant curves and orbits cannot escape.

Idea: Moser's small twist theorem.

Auxiliary Circles

Theorem 2

The auxiliary centers from the construction of the outerArea length billiard about $K$ approximate curves similar to the distant orbits of the outerArea area billiard about $K$.

Proof Sketch

Idea: $\varphi$ ~ {{ "$w(\\theta)/r$" }}, so $\rho$ ~ {{ "$r/w(\\theta)$" }} ~ radial function of $(K+(-K))^*$. However, the center leads orbit point by {{ "$\\pi / 2$" }} in argument.

Conjectures

Conjecture 1

The discontinuity set {{ "$\\Sigma$" }} satisfies {{ "$\\dim_H(\\Sigma)>1$" }} for every polygon.

Conjecture 2

Every polygon admits an unbounded orbit which is dense in the closure of {{ "$\\Sigma$" }}.

Conjecture 3

The square admits a disk of orbits which escape “as fast as possible.” That is, with $|T^n(x)|$ ~ {{ "$\\sqrt{n}.$" }}
