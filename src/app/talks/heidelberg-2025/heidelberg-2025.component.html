Symplectic Billiard Table Map

Question: given a convex $n$-gon $P$, can we construct $Q$ circumscribed s.t. $P$ is a closed symplectic billiard trajectory in $Q$?
Answer: yes!

Symplectic Billiard Table Map

Experimental Results

In general, an $n$-gon tends towards...
  • $n=5$: regularity
  • $n=6$: 3-fold rotational symmetry
  • $n\geq 7$: degeneracy ($k$-gon with {{ '$k\\leq\\left\\lfloor\\frac{n-1}{2}\\right\\rfloor$' }})
Hard to prove: {{ '$\\{n$-gons$\\}/\\mathrm{Aff}$' }} lacks natural coordinates.

Twisted Polygons

Definition:

An (affine) twisted $n$-gon is a sequence of vertices

{{ '$(\\{\\dots,v_{-1},v_0,v_1\\dots\\},M)$' }}

where {{ '$M\\in\\mathrm{Aff}$' }} and {{ '$v_{n+i}=Mv_i$.' }}

Twisted Polygons

Remark:
  • {{ '$\\dim(\\{n$-gons$\\}/\\mathrm{Aff}) = 2n-6$.' }}
  • {{ '$\\dim(\\{$twisted $n$-gons$\\}) = 2n + 6$.' }}
  • {{ '$\\dim(\\{$twisted $n$-gons$\\}/\\mathrm{Aff}) = 2n$.' }}
  • Larger space but no pesky relations!

Result for 2-gons

Theorem (C.–Tabachnikov): When $n=2$, the map is super-integrable: there is an invariant Poisson bracket with three Poisson-commuting integrals:
  1. det($M$)
  2. tr($M$)
  3. a mysterious third one
