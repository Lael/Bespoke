Dynamical Systems

A dynamical system is a collection of measurable quantities which evolve in time according to a set of rules. Examples include:
  • the weather
  • iterated function systems
  • geodesic flow
  • billiards

Selected Results

For the usual outer billiard,
  • certain polygons result in beautiful tilings
  • far-away orbits follow predictable curves
  • the regular pentagon has fractal singularity set
  • unbounded orbits exist around some polygons(!)
